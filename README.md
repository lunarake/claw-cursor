# claw-cursor

[claw-empire](https://github.com/GreenSheep01201/claw-empire)에 Cursor Agent CLI를 provider로 추가하기 위한 통합 가이드입니다.

## 사용법

AI 코딩 도우미(Cursor, Claude, ChatGPT 등)에게 **문서를 읽으라고** 요청하세요. (파일 첨부 불필요)

```
아래 CURSOR_INTEGRATION_LLM_GUIDE.md 문서를 읽고, claw-empire 프로젝트에 Cursor Agent CLI를 provider로 통합해 주세요.
문서에 있는 파일 경로와 변경 내용을 그대로 적용해 주시고, **12~13번 DB 스키마/마이그레이션**을 반드시 적용해 주세요.
```

- **Cursor 사용 시**: `@cursor/CURSOR_INTEGRATION_LLM_GUIDE.md` 로 문서를 참조하도록 하거나, 문서를 열어 읽으라고 요청하세요.
- **문서 위치**: [cursor/CURSOR_INTEGRATION_LLM_GUIDE.md](https://github.com/lunarake/claw-cursor/blob/master/cursor/CURSOR_INTEGRATION_LLM_GUIDE.md)
