# Cursor Agent 통합 가이드

이 폴더는 [claw-empire](https://github.com/GreenSheep01201/claw-empire)에 Cursor Agent CLI를 provider로 추가하기 위한 가이드입니다.

## 사용법

AI 코딩 도우미(Cursor, Claude, ChatGPT 등)에게 아래를 붙여넣고, `CURSOR_INTEGRATION_LLM_GUIDE.md` 를 함께 첨부하세요:

```
아래 CURSOR_INTEGRATION_LLM_GUIDE.md 문서를 읽고, claw-empire 프로젝트에 Cursor Agent CLI를 provider로 통합해 주세요.
문서에 있는 파일 경로와 변경 내용을 그대로 적용해 주시고, **12~13번 DB 스키마/마이그레이션**을 반드시 적용해 주세요.
```
